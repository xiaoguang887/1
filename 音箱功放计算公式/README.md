# 音箱功放搭配计算器

这是一个用于计算音箱与功放最佳搭配方案的网页工具。通过输入音箱参数、房间信息和功放参数，用户可以获得预期的声压级和搭配建议。

## 功能特点

1. **参数输入**：支持输入音箱灵敏度、阻抗、功率范围，房间大小、听音距离，以及功放的阻抗和功率等参数。
2. **实时计算**：点击计算按钮或修改参数后自动重新计算。
3. **结果展示**：显示预期最大声压级(SPL)、功率匹配度、阻抗匹配度、房间适配度和综合评估。
4. **参考图表**：提供3米和3.5米听音距离下的功率参考图表。
5. **搭配指南**：包含音箱功放搭配的基本知识和建议。

## 使用方法

1. 在浏览器中打开网页链接（网络版）或打开 `index.html` 文件（本地版）。
2. 在输入表单中填写您的音箱参数、房间信息和功放参数。
3. 点击"计算搭配方案"按钮或直接修改参数后查看计算结果。
4. 根据计算结果和建议调整您的音箱和功放搭配。

## 文件结构

- `index.html`：网页的主要结构和表单。
- `style.css`：网页的样式表，负责页面的布局和外观。
- `script.js`：实现计算逻辑和结果显示功能。
- `README.md`：项目说明文档。
- `CNAME`（可选）：用于自定义域名（网络版）。

## 技术实现

- 使用HTML5创建页面结构。
- 使用CSS3实现响应式布局和美化界面。
- 使用JavaScript实现计算逻辑和交互功能。
- 无需后端支持，纯前端实现，可直接在浏览器中打开使用。

## 网络版部署指南

由于这是一个纯前端应用，您可以轻松将其部署到各种免费的静态网站托管平台上，让任何人通过链接访问。以下是几种常用的部署方法：

### 方法1：GitHub Pages

1. 在GitHub上创建一个新的仓库
2. 将所有文件上传到仓库
3. 进入仓库的"Settings" > "Pages"
4. 在"Source"选项中选择分支（通常是main或master）和目录（通常是/root）
5. 点击"Save"，几分钟后您的网站将在 `https://<username>.github.io/<repository>` 上可用

### 方法2：Netlify

1. 访问 [Netlify](https://www.netlify.com/) 并注册一个账号
2. 点击"New site from Git"
3. 连接您的GitHub/GitLab/Bitbucket账号
4. 选择包含计算器文件的仓库
5. 点击"Deploy site"，等待部署完成
6. 您的网站将在一个随机生成的Netlify域名上可用，您也可以添加自定义域名

### 方法3：Vercel

1. 访问 [Vercel](https://vercel.com/) 并注册一个账号
2. 点击"New Project"
3. 连接您的GitHub/GitLab/Bitbucket账号
4. 选择包含计算器文件的仓库
5. 点击"Deploy"，等待部署完成
6. 您的网站将在一个随机生成的Vercel域名上可用，您也可以添加自定义域名

## 注意事项

- 计算结果仅供参考，实际效果可能因环境和设备差异而有所不同。
- 建议在选择音箱和功放时，优先考虑阻抗匹配和功率匹配。
- 房间的声学特性也会影响最终的听音效果，计算结果未考虑房间声学处理因素。
